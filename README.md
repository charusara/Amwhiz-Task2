BigBasket 

A simple product listing application built using HTML, Tailwind CSS, and JavaScript. Products are fetched from the DummyJSON API with category filtering and search functionality.

Features
1. Product listing from API
2. Category filter
3. Search products
4. Responsive design

Tech Stack
1. HTML
2. Tailwind CSS
3. JavaScript

API - https://dummyjson.com/products

Run: Open index.html in a browser.
